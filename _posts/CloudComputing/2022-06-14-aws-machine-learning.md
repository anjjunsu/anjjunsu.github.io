## Rekognition
Object detection in images and videos using ML.            
Facial analysis and facial search.          

## Transcribe
Converts speech to text         
Uses a deep learning process called automatic speech recognition    

## Polly
Turn text into lifelike speech using deep learning.     

## Translate
Natural and accurate language translation       

## Lex & Connect
Lex: 
- Same technology that powers Alex
- Automatic speech recognition
- Natural language understadning to recognize the intent of tex, callers
- Helps build chatbots, call center bots            

Connect:    
- Receive calls, create contact flows, cloud-based virtual contact center       
- Can integrate with other CRM systems or AWS

## Comprehend   
Natural Language Processing (NLP)       
Use machine learning to find insights and relationships in text; context, sentiment, etc..      

## SageMaker    
Fully managed service for devs / data scientists to build ML models

## Forecast 
Fully managed that uses ML to deliver highly accurate forecasts     
Use cases: Product Demand Planning, Financial Planning, Resource Planning

## Kendra
Fully managed document search service powered by ML     
e.g.) Extract answers from within a document (txt, pdf, ppt, word, etc.)        
Learn from user interactions/feedback to promote preferred results      

## Personalize
Fully managed ML service to build apps with real-time personalized recommendations      
e.g.) personalized product recommendations/re-ranking, customized direct marketing      
This is the technology used by Amazon.com

## Textract
Automatically extracts text, handwriting, and data from any scanned documents using AI and ML       
e.g.) read credit card, pdf, medical records, or ID
