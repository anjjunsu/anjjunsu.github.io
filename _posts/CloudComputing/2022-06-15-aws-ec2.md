## Elastic Compute Cloud (EC2)
We can configure:
> - OS
> - CPU
> - RAM
> - storage (EBX, EFS, EC2 Instance Store)
> - Network card
> - Firewall rules: security group
> - Bootstrap(launching commands when machine starts) script
