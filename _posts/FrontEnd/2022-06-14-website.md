## Static Website
Static website consists of a series of HTML files.  
Each one HTML represents a physical page of a website. That means each page is a separate HTML file.    

PROS:
> - Easy

CONS:
> - Shared objects needs to be re-written for all individual pages  
>   - e.g.) If there's any change on identical footer, all the pages requires modification

## Dynamic Website  
Dynamic website uses server technologies to dynamically build a webpage when receives a request 

PROS:
> - Easier design updates

CONS:
> - More resources are required
